1. table 사용법

         1) 기본 테이블 만들기 
    
        <table > 
            <tr>
                <th>김기일 </th>  
                <td>30세 <td>
            </tr>
        </table>


        일단 무작정 따라해보자! 다음과 같이 만든다면 모양은 

                

                    

                        김기일
                        30세
                    

                


        위와같이 나온다. th는 테이블 제목이므로 bold체로 나온다

        근데 여기서 만약 칸을 치고 싶다면
 
        css를 살짝 적용해줘야 하는데, 
 
        아까 css를 적용해주는 곳은 어디라고 했지?   답 드래그-> head

        그래 거기에가서 <script>를 선언하고 </script> 닫아주고

        다음과 같이 입력하면 된다. 

        
   
        table {
            border: 1px solid blak;

        }
            
                 

                    

                        김기일
                        30세
                    

                

        과 같은 결과를 얻을 수 있을 것이다.
        

        2) 각 테그 설명 

            table : 표를 구성하는 가장 큰 테그

            tr : 표에서 줄을 구성하는 태그

            th : 제목칸을 만드는 태그 

            td : 일반칸을 만드는 테그
