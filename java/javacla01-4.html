1. 조건문

    1) 특정 조건에 따라 문장의 실행여부
    2) 단일 if
     - 조건식에 따라 문장 실행여부 결정
     - 조건식 판결결과가 true 면 {} 바로 실행
                        false 면 {} 생략함
    3) 기본문법
     - 문장1;
     - if(조건식) {
         문장2; 조건식=true 일때 실행
     } 
     - 문장3; 조건식 =false 인경우 실행
     -> 빵1개 쓰면 체력+10, ~를 해결한다면, 점수+100 등  

    4) 예시
     - 정수형 변수 num의 값이 5보다 크면 "num이" 5보다 크다!" 출력

     - 답:
        if(num > 5) { // 조건식
			// 문장2(조건식 결과가 true 일 때 실행)
			System.out.println("num 이 5보다 크다!"); 
		}
		
		System.out.println("정수 num = " + num); // 문장3

2. if ~ else

    1) 조건식 판별 결과에 따라 서로 다른 문장 실행
    2) 조건식 판별 결과가 true 면 if문 {} 을 실행
                         false 면 else문 {} 을 실행
    3) 삼항연산자와 비슷한 형태 is ~ else가 더 상위 조건문

    4) 기본문법
     - 문장1;
     - if(조건식) {
        문장2; 조건식=true
     } else {
        문장3; 조건식=false
     }
     - 문장 4;
       
    5) 예시
     -int num = 5;
     -정수형 변수 num의 값이 5보다 크면 "num이 5보다크다!"출력
     아니면 "num이 5보다 크지 않다!" 출력
     - 답 :
        if(num > 5) {
			// 조건식 판별 결과가 true 일 때 실행할 문장들... (문장2)
			System.out.println("num 이 5보다 크다!");
		} else {
			// 조건식 판별 결과가 false 일 때 실행할 문장들... (문장3)
			System.out.println("num 이 5보다 크지 않다!");
		}
		
		System.out.println("정수 num = " + num); // 문장4

3. if ~ else if ~ else

    1) 두가지 이상의 조건식을 차례대로 판별
    2) 세가지 이상의 경우를 판별하는 조건문
    3) 기본문법
     -if(조건식1) {
         문장1; 조건식1의 결과가 true
     } else if(조건식2) {
         문장2; 조건식1 =false ,조건식2=true경우
     } else if(조건식n {
         문장3; 조건식1,조건식2=false 고 조건식n의 결과가 true경우
     } else {
         문장x; 모든 조건식 결과가 false일때 실행할 문장들(생략가능)
     }

     4) 예시
     - 정수 num이 5보다 크다 or 5보다 작다 or 5와 같다.
     - 답: 
        if(num > 5) { // num 이 5보다 클 경우
			System.out.println("num 이 5보다 크다!");
		} else if(num < 5) { // num 이 5보다 크지 않고, 작을 경우
			System.out.println("num 이 5보다 작다!");
		} else { // num 이 5보다 크지도, 작지도 않을 경우 = 같을 경우
			System.out.println("num 이 5와 같다!");
		}
		
		System.out.println("num = " + num);

4. 중첩 if

    1) if문 내에서 또 다른 if문을 기술하는 것
    ( else 블록 내에서도 가능)

    2) 기본문법
     -if(조건식1) {
         조건식1이 true일때 실행할 문장들
      if (조건식1-2){
        조건식1이 true이고, 조건 1-2가 true일때,  실행할 문장들
     } else {
         조건식 1이 true, 조건식 1-2가 false 일때 실행
     }
    } else {
        조건식 1이 false 일때 실행할 문장
    }

    3) 예시
     -int score =98;
     -String grade="";
     -정수 0~100사이 학점계산
     아니면 점수입력오류

     - 답 : 
        if(score >= 0 && score <= 100) {
			if(score >= 90) { // 상위 if문에서 100 이하를 판별했으므로 90 이상만 판별하면 됨
				grade = "A";
			} else if(score >= 80) {
				grade = "B";
			} else if(score >= 70) {
				grade = "C";
			} else if(score >= 60) {
				grade = "D";
			} else { 
				grade = "F";
			}
			
			System.out.println(score + "점의 학점 : " + grade);
		} else {
			System.out.println(score + "점 : 점수 입력 오류!");
		}
		
	}
