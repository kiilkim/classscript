1. switch ~ case문


    1) if문과 유사하게 조건에 따른 실행 문장을 달리하는 조건문

    2) if문과 달리 조건식이 정수값, 문자열, enum상수  중 하나의 결과가 와야한다.

    3) case 문에는 조건식 판별 결과와 비교할 값을 사용 범위지정 불가능, 값만사용

    4) 조건식 결과와 일치하는 case 문의 :뒤에 있는 문장을 차례로 실행

    5) case문의 값은 중복될수 없고, 순서상관없음

    6) case문 실행 후 break문을 사용하여 switch ~ case 문을 빠져나갈 수있으며,
    break문이 없는 경우에는 조건과 상관없이 다음 case문 또는 deflaut 문을 모두
     실행함(break,switch case 문이 끝날때까지 실행)

    7) default 문은 if 문의 else 문과 동일한 역할을 수행하며 일치하는 case문이
     없는 경우 default문을 실행(default는 생략 가능)
    
    8) if문은 실행 시점에서 조건을 판별하여 실행할 문장이 결정되지만,
     switch case 문은 번역 시점에 실행할 문장이 결정되므로 실행속도가 if문보다 빠름
     
    그냥 읽기 9) - if문이 switch ~ case 문보다 유연하므로 
    *   switch ~ case 문은 if문으로 100% 전환이 가능하지만
    *   if문은 switch ~ case 문으로 전환이 불가능할 수 있다.
    

    9) 기본문법

     siwtch(조건식) {
        case 값1 :
         //조건식 결과 값1과 일치할 경우 실행
        
        case 값 n
         //조건식 결과가 값n 과 일치하는 경우

         [default: //일치하는 case문이 없는경우]
     }

     // case 뒤에 : 빼먹지말자>

2. 반복문


    1) 특정 조건에 따라 문장 또는 블럭을 반복 실행하는 문

    2) for 문, while 문

    3) for 문은 주로 반복횟수가 정해져 있는 곳

    4) while 문은 주로 반복횟수가 정해져 있지 않는 곳

    5) 초기식, 조건식, 증감식, 실행문 으로 구성
    
    6) for 문
     -초기식, 조건식, 증감식의 위치가 정해져있음
     -기본문법

     for(초기식; 조건식; 증감식){
         //조건식 결과가 true 일때 실행할 문장들
     }

     -hellow world 10번 찍어내기

     -값이 1~10까지 2씩 증가하면서 찍어내기 
     -값 10~1까지 1씩 감소
