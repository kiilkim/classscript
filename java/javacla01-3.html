1. 삼항연산자 

    1) 연산에 참여하는 항이 3개인 연산자
    2) 특정 조건에 따라 다른 값을 실행하는 연산자
    -> if ~ else 문과 동일한 기능을 수행
    -> 2가지 경우의 수(true, false)에 대한 결과를 수행가능

    3) 기본문법
        - 연산식 ? 값1 : 값2
        - 연산식에는 결과값이 boolean 타입(true, false)인 식만 올 수 있다
        - 연산식 판별 결과가 true 일때 값1 부분을 실행하고,
        -            ""     false 일때 값2 부분을 실행.
    
    4) 예제
    - int a = 10;
    system.out.println(a % 2 == 1 ? "짝수": "홀수");
    -> true 일때 system.out.println("짝수"); 문장과 동일한 결과
    -> true 일때 system.out.println("짝수"); 문장과 동일한 결과

2. 삼항연산자의 중첩

    1) 경우의 수가 3가지 이상일때
    2) 삼항연산자 내에서 true 또는 false일 때 실행할 부분에 또 다른
     삼항연산자를 사용하여 그 다음 조건을 판별하는 방법

    3) 기본문법
     연산식1? 값1 : 연산식2 ? 값2 : 값3;
    
    4) 홀수 짝수 판별
     int num = a;
     system.out.println(num%2==0? "짝수" : num%2==1? "홀수":"짝수");

3. 비트 단위 논리연산자 (&, |, ~, ^)

    1) and 연산(&) :두 비트가 모두 1일 때 결과가 1이고, 하나라도 0이면 0이다.
     1100(12)
     0100(4)
     --------and연산
     0100(4)

     2) or 연산(|) :두 비트 중 하나라도 1이면 1이고, 둘다 0인경우 0이다.
     1100(12)
     0100(4)
     --------or연산
     1100(12)

     3) XOR연산(^) :두 비트가 같으면 0, 다르면 1, 베타적논리합
     1100(12)
     0100(4)
     --------^연산
     1000(8)

     4) ~연산 : 현재 비트를 반전 (0->1 ,1->0)
     1100(12)
     --------not연산
     0011(8)

4. 비트단위 시프트 연산자

    1) 비트를 좌측 또는 우측으로 이동시키는 연산자
    2) 좌측으로 이동하는 left shift연산자 : x*2와 동일한 연산수행
     - 우측 빈자리는 0으로 채움
     - 정수 << x 비트 할 경우 "정수 *2^x" 결과와 같다.
    3) 우측으로 이동하는 right shift 연산자 : x/2 와 동일한 연산 수행

    4) 예제
     int i = 1;
     system.out.println("i << 1 = " + (i <<));
     * 0 0000000 00000000 00000000 00000001 (1)
     * ---------------------------------------- 좌측으로 2비트 쉬프트
     * 0 0000000 00000000 00000000 00000100 (4) (1 * 2 * 2)
     */
    
    System.out.println("i << 3 = " + (i << 3));
    /*
     * 0 0000000 00000000 00000000 00000001 (1)
     * ---------------------------------------- 좌측으로 3비트 쉬프트
     * 0 0000000 00000000 00000000 00001000 (8) (1 * 2 * 2 * 2)
     */
    
    System.out.println("i << 30 = " + (i << 30));
    /*
     * 0 0000000 00000000 00000000 00000001 (1)
     * ---------------------------------------- 좌측으로 30비트 쉬프트
     * 0 1000000 00000000 00000000 00000000 (1073741824) (1 * 2 ... 30번 반복)
     */

    System.out.println("i << 31 = " + (i << 31));
    /*
     * 0 0000000 00000000 00000000 00000001 (1)
     * ---------------------------------------- 좌측으로 31비트 쉬프트
     * 1 0000000 00000000 00000000 00000000 (-2147483648) (1 * 2 ... 31번 반복)
     * => 부호 비트가 1이 되므로 음수로 바뀜
     */

    System.out.println("i << 32 = " + (i << 32));
    /*
     * 0 0000000 00000000 00000000 00000001 (1)
     * ---------------------------------------- 좌측으로 32비트 쉬프트
     * 0 0000000 00000000 00000000 00000001 (1)
     * => int형 쉬프트 연산에서 32 이상의 쉬프트 연산은 -32 쉬프트 연산과 같다
     *    즉, 32 비트 쉬프트 = 32 - 32 = 0비트 쉬프트와 동일
     */
    
    System.out.println("----------------------");
    
    i = 8;
    System.out.println("i >> 1 = " + (i >> 1));
    System.out.println("i >> 2 = " + (i >> 2));
    System.out.println("i >> 3 = " + (i >> 3));
    System.out.println("i >> 4 = " + (i >> 4));
    System.out.println("i >> 5 = " + (i >> 5));
