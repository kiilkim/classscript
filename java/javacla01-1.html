        1. 형변환 
         -데이터타입 간의 변환이 일어나는 것
         -기본형 끼리의 형변환과 참조형 끼리의 형변환으로 구분

         1) 묵시적 형변환 (=암시, 자동 형변환)
          -작은 데이터->큰데이터 타입
          -별도의 연산자 사용x, 오류가 발생x

         2) 명시적 형변환 (=강제 형변환)
          -큰 데이터->작은 데이터로 변환
          -반드시 형변환 연산자로 사용하여 우변의 데이터 앞에 작은
          데이터 타입 명시
          -이 때, 원본데이터의 넘침 '오버플로우' 발생할 가능성

        2. 예시

         1) int a1 = 128;  byte a2 =127;
         2) 변환전 a1: a1값, a2: a2값을 문자로 나타내라
         3) a1 = a2 ; 에서는 a2가 a1에 담기는 것이다.
            a1 = (int)a2; 라 해도 되지만 안쓴다.
         4) 변환후 a1: a1 값, a2: a2값을 문자로 나타내라
            같아야 정답이다.

         1) int b1 = 128;  byte b2 =127;
         2) 변환전 b1: b1값, b2: b2값을 문자로 나타내라
         3) b2 = b1; 을 실행하라. 
         4) 뭔가 이상하면 고쳐라 
         5) 고치고 변환후 b1: b1값, b2: b2값을 문자로 나타내라.

         * 우변의 데이터 앞에 작은데이터 타입 이름을 써라!(중요)

         1) int c1 = 10; byte c2 = 0;
         2) 만약에 작은 데이터 타입이 큰데이터타입을 담을 수
         있으면 괜찮다 (예를들어 1.8L우유에 1L도 안들어있었다)

         1. print(), println(), printf()

          1) print(), println() 의 차이는 줄바꿈 여부의 차이 
          2) print() 는 아무데이터 없으면 출력불가

          3) printf()
           -출력할 형식을 미리 지정하여 원하는 형태로 데이터 출력 format
           -""안에 지정문자를 사용하여 형식을 지정한 뒤 ""우측에 
           콤마(,)를 붙이고 실제 데이터 지정


         2. 예시
          1) system.out.println("java");
             system.out.println("programing");

             system.out.print("java");
             system.out.println("programing");

             의 차이

           2) system.out.printf("1234567890");
              system.out.printf("1234567890 \n");
              
              '\n' 사용하여 줄바꿈 필요

           3) %d 또는 %Xd : 10진수 출력 (x부분은 옵션)
           system.out.printf("%d$ \n", 12345);
           system.out.printf("%10d$ \n", 12345); 10자리 확보후 오른쪽/어디쓰임?
           System.out.printf("%-10d$ \n", 12345); 왼쪽정렬/ 당최 어디쓰임?

           4) %f 또는 %Xf : 10진수 실수 출력 (X부분은 옵션)

           System.out.printf("%f \n", 3.141592);
		   System.out.printf("%2f \n", 3.141592);
           System.out.printf("%.2f \n", 3.141592); .x에서 x자리만큼 소수점표시
           
           5) 10진수, 8진수, 16진수 출력

           System.out.printf("%d \n", 255); 10
           System.out.printf("%o \n", 255); 8
           System.out.printf("%x \n", 255); 16진수
           
           6) 8진수 정수 표기법 : 숫자앞에 접두어 0 붙입
           
           System.out.println(0777); 8진수는 7까지!

           7) 16진수 정수 표기법 : 숫자앞에 접두어 0x
           
           System.out.println(0xA);  A=10이므로

           8) %c : 문자 1개 지정

           9) %s 또는 %Xs : 문자열 지정(X 는 옵션)
           System.out.printf("%s \n", "Korea"); // 문자열 출력
           System.out.printf("%10s \n", "Korea"); // 문자열 10자리 확보 후 우측 정렬 출력
           System.out.printf("%10.3s \n", "Korea"); // 문자열 10자리 확보, 3글자만 우측 정렬 출력
           System.out.printf("%10.3S \n", "Korea"); // 문자열 모두 대문자로 출력
           String str = "Java Programming";
           System.out.printf("%S \n", str); // 변수도 사용 가능
