1.매서드

1) 어떤 작업을 수행하는 코드들의 집합 = 작업의 최소단위

2) 선언부와 구현부로 구분하여 정의

3) 반드시 호출해야만 실행될 수 있다. 

4) 간단한 개략도

 - 호출하는 매서드를 caller, 작업수행 매서드를 worker라 함
 - 리턴값이 없을 경우 return타입을 생략할 수 없다. 
 - 반드시 void 라는 특수타입 설정(리턴값이 없다는 의미)

5) 리턴타입은 한가지만 명시, 리턴할 수 있는 값도 한가지만

6) 정의 기본 문법
 [제한자] 리턴타입 매서드명([매개변수...]) {
     //선언부
     구현부 내에 호출할 작업을 기술 
     [return [값];]
 }

 7) 형태에 따른 분류
 - 매개변수도 없고, 리턴값도 없는 메서드
 - 매개변수는 없고, 리턴값만 있는 메서드
 - 매개변수만 있고, 리턴값은 없는 메서드
 - 매개변수도 있고, 리턴값도 있는 메서드
 ----------------------------------------
 - 매개변수가 2개 이상인 메서드(3번 또는 4번 형태 모두 사용 가능)
