1. JAVA의 특징


        1) 독립적이다. WORA (Write Once, Run Anywhere) 


        2) 추상적, 다형성, 캡슐, 상속성 


        3) GC (Garbage Collector)다. 즉, 메모리 할당을 잘함 


        4) 다양한 API(recipe)가 있는데, 많아서 문제가 되기도 함. 신뢰성 확보 


        5) 플랫폼 종속적이다. 


        6) JRE 안에 JVM과 실행 library가 있다. 


        7) 필수로 JDK 를 필요로 한다. eclipse는 편집도구 IDE다.
